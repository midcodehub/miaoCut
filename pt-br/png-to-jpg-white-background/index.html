Dicas para diferentes casos de uso

  • Para documentos e para a maioria dos mercados, opte pelo fundo branco e qualidade de 92% – é o padrão seguro universal.
  • Para plataformas de comércio eletrônico que exigem branco puro (255.255.255), a predefinição Branco é exata em bytes — sem tingimento.
  • Para impressão ou arquivamento, defina a qualidade como 100%. O tamanho do arquivo aumenta, mas o codificador praticamente não produz artefatos de compactação.
  • Para imagens vinculadas à Web em que o tamanho é mais importante do que a perfeição visual, 75-85% reduz drasticamente o tamanho do arquivo, com apenas uma leve perda de qualidade.

Perguntas frequentes

Por que meu PNG fica com um fundo preto quando eu o converto para JPG?

JPG não oferece suporte a transparência. Quando um conversor ingênuo nivela um PNG transparente, ele preenche os pixels alfa com preto por padrão. MiaoCut os preenche com branco (ou qualquer cor que você escolher) para que o resultado fique do jeito que você espera - sem manchas pretas surpresa.

Que cor de fundo devo escolher?

Branco é o padrão seguro para documentos, comércio eletrônico e a maioria dos mercados. Cinza claro funciona bem para fotografia de produtos. O preto combina com designs com temas escuros. O seletor de cores personalizado abrange as cores da marca.

Minha imagem é carregada em um servidor?

Não. Este conversor roda 100% no seu navegador usando Canvas. Sua imagem nunca sai do seu dispositivo – nada é carregado, armazenado ou registrado.

Qual configuração de qualidade JPG devo usar?

92% é um padrão equilibrado – visualmente sem perdas para a maioria das fotos com tamanho de arquivo razoável. Use 100% para impressão ou arquivamento. Use 75-85% se precisar de arquivos menores para a web e não se importar com compactação leve.

Posso converter WebP ou outros formatos também?

Sim. O conversor aceita PNG, WebP e qualquer formato que seu navegador possa decodificar. A saída é sempre JPG.
